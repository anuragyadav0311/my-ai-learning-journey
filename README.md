# My AI Learning Journey

This repository is my personal roadmap for learning Data Science, Machine Learning, Deep Learning, NLP, and MLOps from the ground up. It is organized as a progressive workspace so I can keep notes, practice exercises, reference material, and projects in one place.

## What This Repo Is For

- Building fundamentals first, then moving into applied machine learning and deployment.
- Keeping short notes and useful links in a single repo instead of scattered across different tools.
- Tracking progress through small practice folders and end-to-end projects.

The folders are tracked with `.gitkeep` files so the full learning structure stays visible on GitHub even before each section has content.

## Learning Plan

1. Python basics, control flow, data structures, functions, files, exceptions, and OOP.
2. Data analysis with NumPy, Pandas, Matplotlib, Seaborn, and EDA.
3. Statistics and probability for model intuition.
4. Feature engineering and classical machine learning.
5. MLOps, deployment, and workflow tooling.
6. NLP, deep learning, and end-to-end projects.

## How I Use This Repo

- Add quick notes, code snippets, and practice files inside the topic folders.
- Put curated references in `00-Resources`.
- Keep project work isolated in `10-End-to-End-Projects`.
- Use small, focused folders so each topic is easy to revisit later.
